# 用法介绍
以下是隔夜 / 7 天 Shibor 的异动阈值与常见触发因素速查表（截至 2025 年 12 月 25 日，基于当前市场环境与监管口径），可直接用于快速判断资金面紧张程度：
指标	异动阈值	常见触发因素	监测提示
隔夜 Shibor	1. 正常区间：1.25% - 1.45%
2. 边际趋紧：1.45% - 1.6%（超逆回购利率）
3. 明显紧张：>1.6% 且单日跳升≥20BP
4. 极端紧张：>2% 或盘中触及 3%+	缴税 / 缴准日、跨季 / 年末、政府债大额缴款、央行公开市场净回笼、非银机构头寸集中需求	波动最剧烈，优先看单日跳升幅度与收盘是否持续超阈值
7 天 Shibor	1. 正常区间：1.4% - 1.55%（锚定 7 天逆回购利率 1.4%）
2. 边际趋紧：1.55% - 1.7%
3. 明显紧张：>1.7% 且连续 3 日上行
4. 极端紧张：>2% 且与 OMO 利率偏离≥50BP	跨周 + 跨节资金需求、MLF 到期、同业存单集中到期、短期流动性缺口预期	兼顾即时压力与短期预期，看连续多日趋势更可靠
使用说明
阈值为参考值，需结合央行 7 天逆回购利率（当前 1.4%）、DR001/DR007 等配套指标交叉验证。
极端情形（如 2013 年钱荒、2025 年初短期异动）下，隔夜利率可能突破 2% 甚至更高，需同步关注央行是否加码投放。
月末 / 季末 / 年末等关键时点，阈值可适度上移 5-10BP，因资金面波动常态化
