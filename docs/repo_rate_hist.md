国债逆回购的定盘利率通常指银行间市场的**回购定盘利率**，是基于特定时段交易数据编制的利率基准参考指标，并非交易所国债逆回购的实时成交利率，主要用于银行间市场定价参考。以下是详细介绍：
### 核心定义与编制规则
- **定义**：回购定盘利率是以银行间市场每个交易日上午9:00-11:30的回购交易利率为基础，借鉴国际经验编制的利率基准，每日上午11:30对外发布。其核心品种包括FR001（隔夜）、FR007（7天）、FR014（14天）等，以及银银间的FDR系列（FDR001、FDR007、FDR014）。
- **编制方法**
    |品种|样本范围|计算基础|特殊情况处理|
    | ---- | ---- | ---- | ---- |
    |FR系列（FR001/FR007/FR014）|银行间市场9:00-11:30的全部回购交易（含国债、央行票据、政策性金融债质押）|对应期限（R001/R007/R014）的全部成交利率|无成交时，用对应期限Shibor×365/360替代|
    |FDR系列（FDR001/FDR007/FDR014）|银行间市场9:00-11:30存款类机构的利率债质押回购交易|对应期限（DR001/DR007/DR014）的全部成交利率|无成交时，用对应期限Shibor×365/360替代|

### 与交易所国债逆回购利率的区别
- **适用市场不同**：定盘利率针对银行间市场，是机构间资金拆借的参考基准；交易所国债逆回购利率（如沪市GC001、深市R-001）是个人与机构在交易所市场的实时成交利率。
- **定价机制不同**：定盘利率由中国外汇交易中心计算发布，具有官方基准属性；交易所逆回购利率由市场供需决定，实时波动，受短期资金面影响大。
- **用途不同**：定盘利率常用于金融产品定价、业绩基准等；交易所逆回购利率直接影响投资者的实际收益，是交易时的成交价格。

### 作用与意义
1. **市场参考基准**：为银行间市场的回购交易、债券发行、衍生品定价等提供权威参考，增强市场利率透明度与定价效率。
2. **资金面观测指标**：反映银行间市场短期资金松紧程度，是央行监测市场流动性、制定货币政策的重要参考。
3. **风险管理工具**：帮助金融机构评估利率风险，优化资产负债管理，也为投资者判断市场资金面提供辅助依据。

银行间定盘利率（以**回购定盘利率**为代表）和银行间同业拆借利率（以**Shibor**为代表），均是银行间市场的核心利率基准，但二者在**交易标的、定价机制、适用场景**等方面存在本质区别。

### 核心区别对比表
| 对比维度 | 银行间定盘利率（FR/FDR系列） | 银行间同业拆借利率（Shibor） |
| :--- | :--- | :--- |
| **交易标的** | 以**债券质押式回购交易**为基础，资金融入方需提供利率债等质押品 | 以**无担保的同业信用拆借**为基础，无质押品，依赖交易对手信用 |
| **定价核心** | 反映**质押式资金拆借成本**，受质押券种流动性、市场资金面双重影响 | 反映**信用拆借成本**，核心受银行间信用资质、市场资金松紧影响 |
| **编制主体与时间** | 中国外汇交易中心，每日**11:30**发布，基于当日9:00-11:30成交数据 | 全国银行间同业拆借中心，每日**11:00**发布，基于报价行的报价计算 |
| **样本范围** | FR：全市场机构的利率债质押回购成交<br>FDR：仅存款类机构的利率债质押回购成交 | 由18家报价行（大型银行、股份制银行等）提供报价，剔除最高最低各4家后取算术平均 |
| **期限品种** | 以**短期为主**，主流品种为隔夜（001）、7天（007）、14天（014） | 覆盖**短中长期**，从隔夜到1年共8个期限品种 |
| **风险属性** | 质押交易，**信用风险极低**，利率波动主要反映资金面松紧 | 信用拆借，**隐含交易对手信用风险**，资质较弱的银行拆借利率通常更高 |
| **核心用途** | 债券质押式回购交易定价基准、衍生品（如利率互换）定价参考、央行流动性监测指标 | 同业存单发行定价、贷款利率基准（如LPR挂钩1年期Shibor）、金融产品估值基准 |

### 关键补充说明
1. **本质属性差异**
    - 银行间定盘利率的底层是**“质押融资”**：资金借入方必须用国债、政策性金融债等优质债券做抵押，本质是“以券换钱”，因此利率更贴近**无信用风险的资金成本**。
    - 银行间同业拆借利率的底层是**“信用融资”**：资金借入方仅凭自身信用获得资金，本质是“凭信用借钱”，利率包含**信用溢价**，能反映不同银行的信用资质差异。

2. **与货币政策的关联度**
    - 回购定盘利率（尤其是FDR系列，存款类机构质押回购）**更受央行公开市场操作影响**，是观察央行流动性投放效果的核心指标，比如DR007定盘利率常被视为市场“政策利率锚”的参考。
    - Shibor更多反映**银行体系内的信用融资需求**，受银行自身流动性缺口、信贷投放节奏影响更大，对中小银行的资金紧张程度更敏感。
