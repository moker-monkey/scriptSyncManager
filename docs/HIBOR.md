#### 四、常见触发因素与市场影响
|触发因素|对HIBOR影响|市场传导|
| ---- | ---- | ---- |
|联系汇率干预|弱方触发→HIBOR快速上行；强方触发→HIBOR下行|港元汇率稳定，港股融资成本波动|
|大额IPO/港股分红|短端HIBOR跳升（如大型IPO申购期）|资金短期抽水，港股波动加剧|
|季末/年末缴税|隔夜/7天HIBOR阶段性走高|银行间流动性收紧，信贷定价上行|
|离岸人民币跨境流动|CNH HIBOR波动，与在岸SHIBOR价差扩大|影响跨境贸易结算与CNH债券定价|
|美联储加息/降息|港美利差变化，触发套息交易|HIBOR跟随波动，资金跨境流动加速|

#### 五、每日监测行动指引（3分钟完成）
1.  查价格：11:15后获取港元隔夜/7天HIBOR，11:30后获取CNH对应期限利率。
2.  判阈值：对照“边际/明显/极端”三档，结合均值与标准差快速定性。
3.  看信号：同步核对港元汇率（是否触及7.75/7.85）、银行总结余、金管局操作。
4.  评影响：评估对港股融资成本、房企发债、跨境资金流动的潜在冲击。
5.  做记录：每日归档数据，形成趋势判断，提前预警IPO、缴税等短期冲击。

### 六、使用提示
-  阈值需根据市场环境（如美联储政策、香港流动性）每月更新，建议以近30日滚动数据校准。
-  CNH HIBOR受在岸政策与跨境资本流动影响更大，需结合离岸人民币汇率（CNH/USD）综合判断。
-  金管局贴现窗使用量、外汇基金票据发行情况可作为资金紧张的“确认信号”。
